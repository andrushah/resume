Завдання:

  • - Вивести HTML дерево!
  • - Для виводу використовувати масив з об'єктами. Масив приблизно такого вигляду:
  • const array = [{ name: ‘name1’, url: ’url1’}, { name: ’name2’, url: ’url1’ }, [{ name: ‘name1’ url: ’url1’}, { name: ‘name1’ url: ’url1’}], { name: ‘name3’, url: ’url3’ }, [{name: ‘name1’, url: ’url1’ }, [{ name: ’name2’, url: ’url1’ }]];
  • - Масив записуєтся вручну.
  • - У нас n кількість об'єктів і масивів з об'єктами в середині масиву з об'єктами, уявіть що вам не відомо скільки там всього. Потрібно написати такий код, який обраховує будь яку кількість та будь який рівень вкладеності (РЕКУРСІЯ)
  • - Кожний масив всередині масиву це нова група, яка в браузері має вимальовуватись у вигляді + і списку всередині.
  • - Коли нажимаєте на знак + він стає знаком – і випадає список з категоріями «name: some name, url: some url».
  • - Вигляд дерева: мають бути лінії які супроводжують дерево вкладеності. Всюди де є ще вкладення стоїть +, а де вкладень немає , відповідно немає +. (вкладення відповідно, це масив у масиві, лише в такому випадку повинна вималюватись група із знаком +)
  • - По HTML/CSS, лінії дерева мають бути строго по середині тексту. Якщо елемент останній в списку лінія має закриватися прямим кутом.
  • - Вся інформація в “дерево файлів” виводиться за допомогою JS рекурсивно по масиву даних.
  • - При кліку мишкою на категорію, повинно виводитись попап вікно з назвою вибраної категорії чи групи.
  • - На старті сторінки фокус повинен бути на першій групі в дереві.
  • - Фокус можна зміщувати подіями: нажиманням стрілки вниз та вверх на клавіатурі, коли фокус на категорії тоді вилітає попап з її назвою.
  • - Усі події повинні бути зареєстровані на одному елементі в HTML, це на головному врапері в якому виводиться весь контент.
  • - Якщо відкрилась категорія повинно автоматично переключати в її дерево фокус

Використано такі технології:

  • HTML5
  • CSS3
  • SASS
  • EcmaScript
